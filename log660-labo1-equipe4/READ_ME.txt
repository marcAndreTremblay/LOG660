Notre application est un projet visual studio v.2013

Démarer l'application :
	-Detruire la BD avec les scripte ./SQL/DestructionTable.sql	
	-Lancer le scripte de construction dans le ./SQL/DatabaseBuild.sql	
	-Lancer : ./Bin/LOG660_InsertionProgram.exe (L'éxécution prend environ 10min)
	-Lancer le scripte ./SQL/ProcedureTrigger.sql		
Code Source :
	Le dossier ./LOG660_InsertionProgram contient le code source ouvrable avec visual studio

SQL Scripte:
	Le dosier ./SQL contient le scripte à éxécuter pour destruire et recontruire la BD
